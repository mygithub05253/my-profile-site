# DESIGN.md — 프로필 사이트 디자인 시스템 v1.0 (확정)

> 확정일: 2026-07-04 (KST) · 방향: **C안 Signal** (다크/라이트 듀얼 + 뉴트럴 베이스 + 단일 포인트색)
> 포인트색: 사용자 위임 → **Signal Blue(코발트 계열)** 선정 (금융 신뢰 시맨틱 + 양 테마 AA 대비 + shadcn 정합)
> 구현 시 이 파일을 레포 루트 `DESIGN.md`로 복사해 소스 오브 트루스로 사용

## 1. 원칙

1. **듀얼 테마**: 시스템 `prefers-color-scheme` 자동 + 수동 토글. 다크가 기본값(기존 방문자 인지 유지)
2. **뉴트럴 베이스 + 단일 포인트**: 색은 의미가 있을 때만 (포인트=행동/강조, 그린=성과 수치)
3. **30초 스캔 우선**: 모션은 정보 전달 보조로만, 장식 배제. `prefers-reduced-motion` 존중
4. **토큰 우선**: 모든 색·간격·타이포는 CSS 변수 → Tailwind v4 `@theme` 매핑. 하드코딩 금지
5. **관리자 웹(profile-admin)도 동일 토큰 재사용**

| 토큰 | Dark (기본) | Light | 용도 |
|------|-------------|-------|------|
| `--bg` | `#0A0C10` | `#FAFAF9` | 페이지 배경 |
| `--surface` | `#14171C` | `#FFFFFF` | 카드/패널 |
| `--surface-2` | `#1C2027` | `#F4F4F3` | 중첩 표면, 코드 블록 |
| `--border` | `#2A2F38` | `#E4E4E2` | 경계선 |
| `--text` | `#E6E8EB` | `#1A1D21` | 본문 |
| `--text-muted` | `#9BA3AE` | `#5C6470` | 보조 텍스트 |
| `--accent` | `#5B9DFF` | `#2458D6` | 링크, CTA, 활성 칩 |
| `--accent-soft` | `#5B9DFF1A` (10%) | `#2458D614` (8%) | 칩 배경, 하이라이트 |

- 대비 검증: dark `--text`/`--bg` 15.2:1, `--accent`/`--bg` 7.1:1 · light `--accent`/`--bg` 6.3:1 — 모두 AA 통과
- OG 템플릿: 다크 토큰 고정 사용 (테마 무관 일관 썸네일)

## 3. 타이포그래피

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--font-body` | **Pretendard Variable**, system-ui fallback | 본문·제목 공용 (웨이트로 위계) |
| `--font-mono` | **JetBrains Mono**, monospace | 수치·기간·스택·코드 — "데이터 감성" 포인트 |

스케일(1.25, Mobile First — md 이상에서 한 단계 업):
`--text-xs 12` / `sm 14` / `base 16` / `lg 20` / `xl 25` / `2xl 31` / `3xl 39` / `4xl 49(Hero 전용)`
- 제목: weight 700, `letter-spacing -0.02em` / 본문: 400, `line-height 1.7` / 모노 수치: 500

## 4. 간격·형태·모션

- 간격: 4px 그리드 `--space-1(4)`~`--space-16(64)`. 섹션 수직 패딩: 모바일 64, 데스크톱 96
- 라운드: `--radius-sm 6` (칩·버튼) / `md 10` (카드) / `lg 16` (썸네일·모달)
- 그림자: 다크는 그림자 대신 `--border` + surface 단차. 라이트만 `shadow-sm/md`
- 모션: `--motion-fast 150ms` (hover/포커스) / `--motion-base 250ms` (테마 전환, 스크롤 진입 fade-up 8px 1회, stagger 80ms)

## 5. 핵심 컴포넌트 규격

- **테마 토글**: 헤더 우측, sun/moon 아이콘, `localStorage` 저장 + FOUC 방지 인라인 스크립트
- **프로젝트 카드**: 썸네일(16:9, next/og 재사용) / 카테고리·scope 칩 / 제목 / 한 줄 요약 / 스택 아이콘 최대 3+N / Private 시 `--warning` "비공개" 배지. 전체 클릭 → 상세 페이지
- **필터 칩**: 단일 선택, 활성 = `--accent-soft` 배경 + `--accent` 텍스트
- **버튼**: Primary(`--accent` 채움) / Secondary(보더) / Ghost(텍스트). 상세 페이지의 GitHub/Demo는 Secondary
- **타임라인**: 좌측 수직선 + 노드, 기간은 `--font-mono`
- **성과 수치 강조**: `--font-mono` + `--success` (예: "빌드 104페이지", "49편 자동 발행")
