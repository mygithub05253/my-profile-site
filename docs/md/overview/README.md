# overview/ — 프로젝트 개요서

## 이 폴더에 있는 것

프로젝트 전체(자동화 생태계 + 프로필 사이트 + profile-admin)의 목적·범위·이해관계자·성공 기준을 담은 최상위 개요 문서. 다른 도메인 문서를 읽기 전 진입점.

| 경로 | 내용 |
|------|------|
| `version1.0/` | 프로젝트 개요서 v1.0 (현행 정본) |

## 버전 관리 방식

- `versionX.Y/` 폴더 단위. 개요서는 단일 문서(3종 세트 미적용 — 자체가 요약 성격)
- 프로젝트 범위·목표가 바뀌는 큰 변곡점에서만 버전업 (세부 스펙 변경은 각 도메인 문서 소관)
