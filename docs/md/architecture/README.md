# architecture/ — 시스템 아키텍처 설계서

## 이 폴더에 있는 것

4-레포 자동화 생태계(content-hub · velog-backup · my-profile-site · mygithub05253) + profile-admin(예정)의 시스템 구조 정본.

| 경로 | 내용 |
|------|------|
| `version1.0/` | 시스템 아키텍처 설계서 v1.0 (현행 정본) — 전체 구성도, 시퀀스 다이어그램 4종, 보안 설계(경량 STRIDE), 장애 시나리오 6종·복구 정책, **ADR 11건** |

## 버전 관리 방식

- `versionX.Y/` 폴더 + 3종 세트(설계서 · 요약본 · 변경사항). CN 누적 → 마일스톤 후 일괄 버전업
- **ADR(아키텍처 결정 기록)은 추가만 하고 수정하지 않는다** — 결정이 뒤집히면 새 ADR로 대체(supersede)하고 구 ADR에 표기
- 시퀀스·구성도는 Mermaid 텍스트로 관리 (렌더링 도구 독립성)
- E2E 검증 중 발견된 결함은 장애 시나리오(F-*)로 승격해 복구 정책과 함께 기록
