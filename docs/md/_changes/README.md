# _changes/ — 횡단 검토·변경 보고서

## 이 폴더에 있는 것

특정 도메인에 속하지 않는 **횡단(cross-cutting) 검토 보고서**. 도메인별 변경 이력은 각 `versionX.Y/`의 변경사항 파일이 담당하고, 여기는 여러 문서를 한꺼번에 대조·검증한 결과를 둔다.

| 파일 | 내용 |
|------|------|
| `문서검토보고서_2026-07-04.md` | 16문서 전체 + as-built 코드 대조 검토 — 결함 F 10건(전건 반영), 사용자 결정 D 4건(전건 확정), 리서치 R 4건(전건 반영) |

## 관리 방식

- 파일명에 날짜(`_YYYY-MM-DD`) 포함 — 버전 폴더 미적용, 시점 스냅샷으로 보존
- 보고서의 F/D/R 항목이 해소되면 **보고서 원문은 수정하지 않고** 각 도메인 문서와 마스터인덱스에 결과를 반영한다 (보고서는 감사 이력)
