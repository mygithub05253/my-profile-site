## 관리 방식

- 프롬프트는 버전 접미사(`_vN`)로 관리. 실행 결과는 파일로 남기지 않고 **각 도메인 문서에 직접 반영** — 반영 위치는 마스터인덱스 "리서치 완료 (R)" 표 참조
- 리서치 답변의 수치·API 계약은 반영 시점 기준이므로, 구현 시 공식 문서로 재확인할 것
